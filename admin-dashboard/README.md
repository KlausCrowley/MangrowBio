Admin dashboard lives at:
- UI routes: `src/app/admin/*`
- Auth: `src/app/api/auth/*`, `src/lib/auth.ts`, `src/proxy.ts`
- Admin-protected APIs: CMS and contact management endpoints under `src/app/api`

Main routes:
- `/admin/login`
- `/admin/dashboard`
- `/admin/cases`
- `/admin/blog`
- `/admin/contacts`
